# 移动应用开发接口
# 1.用户登录
# '/user/login'
```
POST /user/login
```

## 请求参数

|   参数名    | 类型 | 必填 | 说明  |
|:--------:| :----: | :----: |:---:|
|  phone   | String | 是 | 手机号 |
| password | String | 是 | 密码  |

## 返回参数

| 参数名  | 类型      | 是否必须 | 备注                           |
| ------- | --------- | -------- | ------------------------------ |
| code    | number    | 必须     | 响应码，1 代表成功，0 代表失败 |
| msg     | string    | 非必须   | 提示信息                       |
| data    | object[ ] | 必须     | 返回的数据                     |
